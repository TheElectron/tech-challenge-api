API Operacional

Esta é uma API RESTful para consulta de dados de livros, populada através de web scraping. Explore os endpoints abaixo ou acesse a documentação completa.

Acessar Documentação da API

Listar Livros

Acesse a lista completa e paginada de todos os livros disponíveis. Ex: /api/v1/livros

Filtrar por Preço

Encontre livros dentro de uma faixa de preço específica. Ex: /filter/price?min=10&max=30

Ver Categorias

Obtenha uma lista de todas as categorias de livros existentes. Ex: /api/v1/categories
